Pulse — Privacy Policy

Last updated: July 2026

Pulse is a personal, single-user application built and operated by its sole user for private, non-commercial use.

Data collected

With the user's explicit OAuth consent, Pulse reads the user's own WHOOP data: recovery, sleep, workout, physiological cycle, and body measurement information.

How data is used

Data is used solely to generate daily training and nutrition recommendations for the user. It is stored in a private database (Supabase) accessible only to the user.

Data sharing

No data is shared with, sold to, or made accessible to any third party. The application has no other users.

Retention and deletion

Data is retained only for the user's personal tracking. All stored data can be deleted at any time, and access can be revoked via the WHOOP app or by revoking the OAuth token, at which point Pulse retains no ability to access WHOOP data.

Contact

For any questions regarding this policy, contact the operator of this application via the GitHub repository hosting this page.
